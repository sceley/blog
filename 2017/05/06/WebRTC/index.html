WebRTC

前端
  • HTML5

序言

目前Web世界中的一个重大挑战是音频和视频的实时通信，这个API是为了实现这个功能。

媒体设备getUserMedia

let promise = navigator.mediaDevices.getUserMedia(constraints);
promise
.then((stream) => {})
.catch((err) => {});
//constraints： {audio: , video:}

以下表示偏好的1280*720摄像机分辨率

{
    audio: true,
    video: {
        width: 1280, 
        height: 720
    }
}

以下要求最低分辨率为1280*720:

{
    audio: true,
    video: {
        width: {min: 1280},
        height: {min: 720}
    }
}

移动设备:

前置摄像头{
    audio: true,
    video: {facingMode: "user"}
}
后置摄像头{
    audio: true,
    video: {facingMode: {exact: "enviroment"}
}

example:

let constraints = {audio: true, video: {width: 1280, height: 720}}
navigator.mediaDevices.getUserMedia(constraints)
.then((mediaStream) => {
    let video = document.getElementById("video");
    video.srcObject = mediaStream;
    video.onloadedmetadata = (e) => {
        video.play();
    };
})
.catch((err) => {
});

WebAudioAPI

工作流程

  1. 创建音频上下文
  2. 在上下文中，创建音频源，如>audio&It;
  3. 创建效果节点，如混响，双二进制，平板，压缩
  4. 选择音频最终目的地, 例如系统扬声器
  5. 将源连接到效果，并将效果连接到目标

创建音频上下文

let AudioContext = window.AudioContext || window.webkitAudioContext;
let audioCtx = new AudioContext();

创建音频源

//通过javascript直接从音频节点(如振荡器)生成
AudioContext.OscillatorNode()
//从原始PCM数据创建，音频上下文具有解码支持的音频格式的方法
AudioContext.createBuffer();
AudioContext.createBufferSource();
AudioContext.decodeAudioData();
//通过HTML标签
AudioContext.createMediaElementSource();
//通过麦克风或摄像头获取
AudioContext.createMediaStreamSource();
//提供简单音调的震荡器音频源
let oscilator = audioCtx.createOscilator();
//控制音量
let gainNode = audioCtx.createGain();

将音频节点和目标节点连接在一起(connect方法)

ossillator.conect(gainNode);
gainNode.connect(audioCtx.destination);

连接多个节点

source = audio.createMediaStreamSource(stream);
source.connect(analyser);
analyser.connect(distortion);
distortion.connect(biquadFilter);
biquadFilter.connect(concvolver);
concvolver.connect(gainNode);
gainNode.connect(audioCtx.destination);

MediaStreamRecordingAPI

该MediaStream记录API可以很容易地录制音频和视频流。使用时navigator.mediaDevices.getUserMedia()，它提供了一种从用户输入设备进行记录的简单方法，并将结果立即用于Web应用程序。音频和视频可以单独或一起记录

捕捉媒体流

let mediaRecorder = new MediaRecorder(stream);
MediaRecorder.start()
//进行记录
MediaRecorder.state
//属性将返回"录制值"
MediaRecorder.ondataavailable = (e){
    buffer = e.data
}
//收集音频数据
start(time)控制时间段或者MediaRecoder.stop()停止录制
MediaRecorder.stop
//停止录制
MediaRecorder.onstop = (e) => {
}
//完成我们的BLOB
