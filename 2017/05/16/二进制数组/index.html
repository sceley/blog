二进制数组

前端
  • HTML5

序言

二进制数组是Javascript操作二进制数据的一个接口

二进制数组由3类对象组成：

  1. ArrayBuffer对象: 代表内存中的一段二进制数据，可以通过’视图’进行操作。
  2. TypedArray视图： 共包括9种类型的视图。
  3. DataView视图： 可以自定义复合格式的视图，比如第一字节是Unit8(无符号8位整数)、第二第三字节是Int16(16位整数)等，还可以是自定义字节序。

简言之ArrayBuffer对象是代表原始的二进制数据，TypedArray视图用于读写简单类型的二进制数据，DataView视图用于读写复杂类型的二进制数据。

TypedArray视图支持的数据类型一共有9种(DataView视图除支持Uint8C以外的其他8种)

数据类型        字节长度        含义                          对应的C语言类型
Int8        1            8位带符号整数                  signed char
Uint8        1             8位不带符号整数              unsigned char
Uint8C        1            8位不带符号整数(自动过滤溢出)unsigned char
Int16        2            16位带符号整数              short
Uint16        2            16位不带符号整数              unsigned short
Int32        4            32位符号整数                  int
Uint32        4            32位不带符号整数              unsigned int
Float32        4            32位浮点数                  float
Float64        8            64位浮点数                  double

ArrayBuffer对象代表储存二进制数据的一段内存, 它不能直接读/写，只能通过视图读/写

ArrayBuffer也是一个构造函数，可分配一段可以存放数据的连续内存区域

let buf = new ArrayBuffer(32);
//生成了一段32字节的内存区域，每个字节的默认值都是0。参数是所需的内存大小(单位为字节);

创建DataView视图，需要提供ArrayBuffer对象实例作为参数。

let buf = new ArrayBuffer(32);
let dataview = new DataView(buf);
dataView.getUint8(0); //0
//以不带符号的8位整数格式读取第一个元素

TypedArray视图,他不是一个构造函数，而是一组构造函数

let buffer = new ArrayBuffer(12);
let x1 = new Int32Array(buffer);
x1[0] = 1;
let x2 = new Uint8Array(buffer);
x2[0] = 2;
x1[0] //2

TypedArray视图的构造函数除了接收ArrayBuffer实例作为参数，还可以接收普通数组作为参数，直接分配内存生成底层的ArrayBuffer实例，同时完成对这段内存的赋值。

let typedArray = new Uint8Array([0, 1, 2]);
typedArray.length //3
typedArray[0] = 5;
typedArray//[5, 1, 2]

ArrayBuffer.prototype.byteLength

返回所分配的内存区域的字节长度

let buffer = new ArrayBuffer(32);
buffer.byteLength//32

如果要分配的内存区域很大，有可能分配失败(因为没有那么多连续的空余内存),所以有必要检查是否分配成功。

if (buffer.byteLength === n){
    //成功
} else{
    //失败
}

ArrayBuffer.prototype.slice()

允许将内存区域的一部分复制生成一个新的ArrayBuffer对象

let buffer = new ArrayBuffer(8);
let newBuffer = buffer.slice(0, 3);

ArrayBuffer.isView()

let buffer = new ArrayBuffer();
ArrayBuffer.isView(buffer)//false
let v = new Int32Array(buffer);
ArrayBuffer.isView(v);//true

TypedArray视图一共包括9种类型

Int8Array： 8位有符号整数，长度位为1个字节
Uint8Array： 8位无符号整数，长度位为1个字节
Uint8ClampedArray： 8位无符号整数，长度位为1个字节, 溢出处理不同
Int16Array： 16位有符号整数，长度位为2个字节
Uint16Array： 16位无符号整数，长度位为2个字节
Int32Array： 32位有符号整数，长度位为4个字节
Uint32Array： 32位无符号整数，长度位为4个字节
Float32Array： 32位浮点数，长度位为4个字节
Float64Array： 64位浮点数，长度位为8个字节
//它们很像普通数组

TypedArray(buffer, byteOffset, length?)

第一个参数(必须):视图对应的底层ArrayBuffer对象。

第二个参数(可选):视图开始的字节自序, 默认从0开始

第三个参数(可选):视图包含的数据个数，默认直到本段内存区域结束

DataView视图

DataView视图本身也是构造函数，接受一个ArrayBuffer对象作为参数生成视图

DataView(ArrayBuffer[, 字节起始位置[, 长度]]);

let buffer = new ArrayBuffer(24);
let dv = new DataView(buffer);

读取                    写入
getInt8                setInt8
getUnit8            setUint8
getInt16            setInt16
getUint16             setUint16
getInt32            setInt32
getUint32            setUint32
getFloat32            setFloat32
getFloat64            setFloat64

Blob对象

Blob对象有两个属性，size属性表示一个Blob对象的字节长度，type属性表示Blob的MIME类型

创建一个代表原始二进制数据的Blob对象

let blob = new Blob([blobParts, type]);

存放任意数量个以下类型的对象
ArrayBuffer
ArrayBufferView
Blob
String

let blob = new Blob(['1234' + '5678'], {type: 'text/plain'});

Blob对象的slice方法

let newBlob = blob.slice(start, end, contentType);

FileReader对象

  1. readerAsBinaryString(读取为二进制字符串)
  2. readerAsText(读取为文本数据)
  3. readerAsDataUrl(读取为DataURL)
  4. readerAsArrayBuffer(读取ArrayBuffer对象)
  5. abort(中断读取操作)
